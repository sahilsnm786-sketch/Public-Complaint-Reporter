Public Complaint Reporter

## Overview
Public Complaint Reporter is a simple HTML-based web application that allows citizens to report public issues such as road damage, water leakage, garbage collection problems, drainage issues, broken footpaths, and other civic complaints.

This project is designed to demonstrate how technology can help citizens communicate with local authorities in an easy and efficient way.

## Features
- Report public complaints
- User-friendly complaint form
- Multiple complaint categories
- Emergency contact information
- Simple and responsive interface
- Email submission using FormSubmit

## Technologies Used
- HTML5
- Basic CSS (Inline Styling)
- FormSubmit

## Complaint Categories
- Road Damage
- Water Leakage
- Garbage Collection
- Street Light
- Drainage Problem
- Broken Footpath
- Other

## Emergency Contacts
- Police – 112
- Ambulance – 108
- Fire Brigade – 101
- Municipal Office
